Stateful pretty-printing of concrete Beluga signatures to HTML.

val pp_signature_to_files : directory:Stdlib.String.t -> Synext.signature -> Stdlib.Unit.t

pp_signature_to_files ~directory signature pretty-prints signature to ".html" files in directory. Each file in signature is pretty-printed to a corresponding file with the same directory structure as signature.

val pp_signature : Stdlib.Format.formatter -> Synext.signature -> Stdlib.Unit.t

pp_signature ppf signature pretty-prints signature to ppf as an HTML page.
